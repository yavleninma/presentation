Страница **v2 — Redesign** содержит переработанные экраны и демо-слайды.

### Экраны (6 шт)

| # | Экран | Роль |
|---|-------|------|
| 01 | Start | Тёмный фон (#111), центрированное поле ввода, минимальный текст |
| 02 | Chat | Полноценный чат: аватарки, бабблы, suggestion chips, кнопка «Собрать черновик» |
| 03 | Pattern Selection | Три паттерна (Питч / Отчёт / Обучение), рекомендованный подсвечен |

### Сквозной сценарий — питч Вняtно

Пользователь создаёт питч-презентацию о сервисе Вняtно для инвесторов.

**Start:** пользователь вводит «Нужно собрать питч для инвесторов»
**Chat:** система уточняет аудиторию и цель → «Хватает для черновика. Собрать?»
**Pattern Selection:** рекомендован «Питч» (Проблема → Решение → Результат → CTA)
**Building:** 4 шага: анализ задачи → подбор структуры → генерация контента → оформление
**Editor:** 6 слайдов, активный — «Три шага до черновика»

|---|-------|------------|

### Ключевые дизайн-решения v2

**1. Визуальный язык**
- Белые панели, мягкие тени (0 4 20 rgba(0,0,0,0.08))
- Акцент: #4568F6 (indigo) — ссылки, кнопки, акцентные элементы
- Зелёный: #2E8C59 — прогресс, галочки, рекомендации, accent bars на слайдах
- Тёмный start: #111111 — контраст для входа
- Шрифт: Inter (Bold/SemiBold/Medium/Regular) — единый на все экраны
- Радиусы: 20px (основные панели), 16px (карточки), 12px (мелкие блоки), 10px (бейджи)

**2. Chat**
- Аватарки с инициалами (серый бот / индиго пользователь)
- Бабблы: бот слева (#F7F8F9), пользователь справа (rgba indigo 10%)
- Suggestion chips с иконками перед текстом
- Зелёная кнопка «Собрать черновик» появляется в потоке

**3. Pattern Selection**
- Три карточки одного размера с иконкой, названием, описанием, списком шагов
- Рекомендованный паттерн: зелёная обводка + бейдж «Рекомендуем»
- Кнопка «Продолжить» под карточками

**4. Editor**
- Топбар: логотип + название документа
- Slide rail слева: миниатюры с номерами, активная с индиго-обводкой
- Canvas: слайд 16:9 с контентом
- Drawer справа: AI-действия (Упростить / Добавить цифры / Сократить) + заметки
- Футер: навигация «Слайд N из 6»

**5. Демо-презентация**
- Тексты про задачу пользователя, без внутреннего словаря
- Каждый слайд самодостаточен и может быть показан руководителю
- Зелёная accent bar сверху каждого слайда — единый визуальный якорь
