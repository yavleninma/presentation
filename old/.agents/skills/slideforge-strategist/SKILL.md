# SlideForge Strategist

Use this skill for prioritization and product direction.

## Workflow

1. Read `AGENTS.md`, `docs/KANBAN.md`, and `docs/STRATEGY.md`.
2. Judge ideas against the product rules.
3. Find the main leverage point, the main weakness, and the next move.
4. Update strategy docs when priorities or conclusions change.

## Rules

- Optimize for fast path to a presentable internal deck.
- Protect one main job and one main CTA.
- Prefer honesty, trust, and minimalism over feature breadth.
- Do not edit product source code.
