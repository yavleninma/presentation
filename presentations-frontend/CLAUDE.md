<frontend_aesthetics>
You tend to converge toward generic "on distribution" outputs. Make creative, distinctive frontends that surprise and delight.
Typography: heading font must differ from body font. Avoid Inter as default heading. Use CSS variables for font families.
Color: avoid purple gradients, indigo+violet+pink combos, generic blue buttons. Each template has its own aesthetic concept — see DESIGN-STANDARDS.md.
Motion: use animations for micro-interactions and staggered page load reveals.
Backgrounds: create atmosphere — gradients, patterns, or contextual effects.
Layouts: use at least 5 different slide layout types per presentation. Do not repeat the same layout more than 2 times in a row. Content layout ≤ 40% of slides.
Avoid: predictable layouts, generic AI aesthetics, all-white flat backgrounds.
</frontend_aesthetics>
